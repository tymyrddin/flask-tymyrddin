IPA project

A project sponsored by /ut7 for supporting Intimate Partner Abuse survivors in various ways:

  • Keeping up with the digital tracking threat landscape
  • Asking survivors as the domain experts for developing up-to-date guides on detecting and removing tracking attempts.
  • Designing and facilitating workshops which include hands-on exercises on tracking, and roleplays on dealing with law enforcement in local contexts.
  • Developing labs for shelters to do mobile testing and forensics using open source tools such as the PiRogue suite.
  • Supporting interested parties in using the lab.

We work closely with health professionals. Measurements on success is going to be hard, but we could use MOVERS.
